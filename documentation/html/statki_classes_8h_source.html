8 /*
9 Assumptions:
10  - Each field can have a pointer to only one ship
11  - Each player has a different ID (string)
12  - The game board's size is 10x10. The Fields are numbered from 0 till 9
13  - A ships' position on the game board is defined by its starting position (uppermost segment or the first segment on the left side)
14  and its orientation. So if lenght=3, orientation=horizontal and pos=(1,1), then the ship is placed on the following fields: (1,1), (1,2), (1,3)
15  - A player cannot shoot the same field more than once (implemented in the client)
16  - A player can only shoot when it's their turn ( implemented by client)
17  - The player can shoot only fields with indexes between 0 and 9 (checked by client)
18 */
